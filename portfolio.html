BLOG PESSOAL

O projeto Blog Pessoal é um exemplo de aplicação fullstack voltado para a temática dos organismos aquáticos. Desenvolvido utilizando as ferramentas MySQL, Spring e React, este projeto permite aos usuários criar, editar e compartilhar postagens relacionadas a aquários, peixes ornamentais, cuidados com a água e outros temas relacionados ao universo aquático. Disponível em: https://github.com/MatheusVSOliveira/Projeto-blogpessoal-Spring_React

REDE SOCIAL

A rede social Tales é um projeto integrador realizado em grupo na Generation Brazil, inspirado na ODS 6 da ONU (Água potável e Saneamento). Seu objetivo é criar uma plataforma interativa para conectar pessoas com interesses relacionados a este tema. Foi desenvolvido utilizando as ferramentas MySQL, Spring e React. Disponível em: https://github.com/MatheusVSOliveira/ProjetoIntegrador-GenerationBrazil-Spring_React

APLICATIVO DE EVENTOS

O Aplicativo de Eventos é um projeto de cadastro de eventos MVC, criado utilizando as ferramentas Spring Boot, MySQL, HTML, CSS e Materialize. Esta aplicação permite aos usuários pesquisar, cadastrar e gerenciar eventos de forma eficiente e intuitiva. Disponível em: https://github.com/MatheusVSOliveira/SpringBoot_EventosApp
